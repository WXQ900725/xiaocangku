1. 安装npm
2. 通过npm安装remixd
3. 创建共享文件夹，使用remixd命令开放连接给remix-ide


具体操作：
1. 安装nodejs, 下载node.msi

https://mirrors.tuna.tsinghua.edu.cn/nodejs-release/v9.9.0/

2. 安装remixd（使用power shell管理员权限）

```
 npm install -g  @remix-project/remixd
```

3. 创建一个文件夹：F:\Remix-Share，然后共享这个文件夹（使用power shell管理员权限）：

```
remixd -s  E:\work\solidity-share --remix-ide https://remix.ethereum.org
```
