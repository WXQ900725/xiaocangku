> 投票

```
$ clique.status()
这是一种调试方法，它返回有关最后 64 个块的签名者活动的统计信息。返回的对象包含以下字段：
inturnPercent: 依次签名的区块百分比
sealerActivity: 包含签名者地址和他们签名的块数的对象
numBlocks: 分析的块数
```

```
$ clique.propose("0xc63cc046a2e7e40bfc1e071f7557d058346b0098", true)
添加签名者将尝试通过的新授权提案。如果 auth参数为真，则本地签名者投票支持将给定地址包含在授权签名者集中。auth设置为，false投票反对地址。
```

```
$ clique.discard("0xc63cc046a2e7e40bfc1e071f7557d058346b0098")
此方法丢弃当前正在运行的提案。签名者不会对该地址投进一步的投票（赞成或反对）
```

> 获取当前投票节点

```
$  clique.proposals
```
